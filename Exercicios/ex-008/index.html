Principais formatações

negrito & Destaque

Negrito: termo em negrito
usando a tag <b> (não semantica)

Destaque: termo em Destaque
usando a tag <STRONG> (semantica)

Italico & ênfase

Italico: Termo em italico
usando a tag <i> (não semantica)

Ênfase: Termo em ênfase
usando a tag <EM>

Texto marcado

Texto marcado: Termo marcado
Usando a tag <MARK>

Texto marcado em vermelho: Vermelho
Usando a tag <MARK>, adicione "style" e depois "background-color"

Texto Grande e Pequeno

Texto grande e Texto pequeno
é importante notar que <big> está obsoleto, o texto pequeno usa a tag <small>

Riscado

Riscado: Termo só que riscado
usando a tag <del>

Inserido & Sublinhado

Inserido: Texto inserido
usando o tag <ins>

Sublinhado: Texto sublinhado
usando o tag <u>

Sobrescrito

Texto Sobrescrito: Sobrescrito
usando o tag <sup>

Subscrito

Texto Subscrito: subscrito
usando o tag <sub>
